# User Flows: Employee

## 1. Onboarding
1.  **Accept Invite:** Click link in email.
2.  **Set Password:** Define secure password.
3.  **Profile Completion:** Fill in personal details (Address, Emergency Contact).
4.  **IBAN Entry:** Enter IBAN for salary (Encrypted upon save).

## 2. Leave Requests
1.  **Dashboard:** View available balance (Wettelijk / Bovenwettelijk).
2.  **Request:**
    -   Select Dates.
    -   Add Reason (Optional).
    -   Submit.
3.  **Notification:** Manager receives email.
4.  **Result:** Approved/Denied notification.

## 3. Sickness Reporting
1.  **Report:** Click "Ik ben ziek" (I am sick).
2.  **Details:** Select start date (default: today). *Medical details are NOT asked (Privacy).*
3.  **Recovery:** Click "Ik ben weer beter" (I am recovered) upon return.
    -   System calculates total days.
    -   If > X weeks, specific flows trigger.

## 4. Documents
1.  **Payslips:** View/Download monthly payslips.
2.  **Contract:** View current contract details.
