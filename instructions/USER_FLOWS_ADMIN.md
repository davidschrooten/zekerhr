# User Flows: Admin

## 1. Onboarding (First Run)
1.  **Login:** Super Admin logs in with temporary credentials/invite link.
2.  **Setup:** Configures Company Name, Address, Fiscal ID (Loonheffingennummer).
3.  **Invite:** Invites the first HR Admin (if different from Owner).

## 2. Employee Management
1.  **Invite Employee:**
    -   Enter Email, Full Name.
    -   Select Role (`employee`, `manager`).
    -   System sends Invite Email.
2.  **Contract Setup:**
    -   Admin enters Start Date, Salary, FTE.
    -   System calculates initial Leave Balance (Wettelijk/Bovenwettelijk).

## 3. Compliance Management
1.  **Poortwachter Dashboard:**
    -   View alerts for Week 6, Week 42, etc.
    -   Upload required documents (PvA).
2.  **WKR Check:**
    -   Review current "Vrije Ruimte" usage.
    -   Approve/Categorize pending expenses.

## 4. Payroll Run
1.  **Review:** Check mutations for the current month.
2.  **Export:** Click "Generate Payroll Export".
3.  **Confirm:** Acknowledge security warning.
